Conceito de Repetição

  • A ideia é repetir uma ação até atingir uma condição;
  • Estrutura de repetição condicional: ou seja, essa repetição aconteça se atender determinada condição;
  • Ao invés de repetirmos o mesmo código manualmente várias vezes, criamos um statement que fará uma checagem em cada loop;
  • Também chamada de lopp;

Estrutura de repetição: while (enquanto)

  • Fará uma ação, enquanto a condição seja atingida;
  • Importante determinar um fim para não criar um loop infinito;

strutura de repetição: do while

  • Semelhante ao while, porém a estrutura muda, pois ele primeiro executa o código, e depois analise se a condição foi atendida;
